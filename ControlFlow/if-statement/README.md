# If Statement

日常生活中我們很常根據條件，做出判斷。比如說：

* 如果下雨，我就不出門。
* 假設 Peter 要出席，我就出席，否則我就不去了。
* 在進行一場球賽的時候，硬幣骰到正面，我先進攻; 反面，你先進攻。

這樣子的邏輯要怎麼在 Swift 中實現了，就要靠 `if-statement`

讓我們來看一下 Swift 的 if 語法，
