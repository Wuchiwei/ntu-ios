# Framework

### 安裝 CococaPods

打開 Terminal，輸入以下指令：

`sudo gem install cocoapods`

[Cocoapods 官網](https://cocoapods.org)

### Create Podfile

在 Terminal 中，前往 Project 所在路徑，輸入以下指令，會產生一個名為 `Podfile` 的新檔案：

`pod install`

### 打開 Podfile 編輯
